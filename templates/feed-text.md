# 闲鱼推广动态文案

## 写法公式

```
开头钩子（1句，引发好奇）→ 中间展开（3-5句，讲做了什么/怎么做的）→ 结尾CTA（1句，引导看主页）
```

## 模板 A：经验分享型

搞了个{什么东西}，分享一下。

我花了{时间}，把{什么}和{什么}彻底打通了。

现在{一句话说效果}。

全程{一句话说过程}：
- {亮点1}
- {亮点2}
- {亮点3}

说实话最难的倒不是{技术难点}，是{真正难的地方}。踩了好多坑，{列举1-2个}……一个个趟过来的。

现在已经稳定跑了{多久}，{数据}。

我把整套流程整理成了教程。需要的可以看看我主页。

## 模板 B：清单推荐型

断断续续折腾了{时间}，把这些东西全串起来了，分享给同样在摸索的朋友。

我现在的{工作流名称}是这样的：

- {工具1} 管{用途1}，{效果}
- {工具2} 做{用途2}，{效果}
- {工具3} 让{用途3}，{效果}

从{起点}到{终点}，{时间}。

把这{数量}个环节整理成了教程。需要的看主页～

## 模板 C：痛点共鸣型

你是不是也遇到过这些情况：
- {痛点1}
- {痛点2}
- {痛点3}

我以前也是，直到我{用了什么方法/工具}。

现在{成果描述}。

整理了一份完整教程，从{起点}到{终点}全覆盖。主页自取。
